Notify
VueJs / NodeJs / Twilio / MongoDB

Notify is a dynamic checklist that features SMS notifications as well as two way SMS messaging. Twilio SMS allows users to add, delete, and update tasks through a custom chatbot. A Node-Cron scheduled function integrates with Twilio to send the notifications to the user.
