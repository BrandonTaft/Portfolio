My Work

React Autocomplete Input Package

An accessible, customizable and easy-to-implement autocomplete input component for ReactJS. It is flexible enough to be used with just a simple array of strings and a single event handler or make it as robust as you want with nested objects and props that give the user complete control over input and output handling and display. Another feature is built in error-handling that can also be easily customized at each step. The component is ready to go right out of the box but also comes with complete access to styling for every element.

CSVisuals

CS Visuals is a series of tutorials that use data visualization to teach common sorting algorithms that are used in computer science. It Currently covers four different algorithms that are thought to be the best for a beginner to learn and features customizable interactive visualizations that enable the user to walk through each step at their preferred speed. The app was built with Next.js and Javascript along with CSS were used for the algorithms and DOM manipulation to create the animations. Lighthouse scores for this app are 98 for performance and 100 for SEO, accessibility, and best practices which helps to provide a nice clean experience for the end user.

Notify

Notify is a mobile conferencing app built with React Native and Expo for client development. The client uses JWT for authentication, react-navigation for navigation, react-native-reanimated for gesture handling and socketIO is used for client-side channel communications. A Node server is used for JWT middleware and for the socketIO server to support communication between clients and the server. A Mongo DB database is used to securely store user information and contain data for messages and push notifications which are sent by a chron job created with node-cron.

Are You Smarter Than Erin

I was tasked with creating a game that would help my peers be more engaged with monthly E-learnings. Erin is my employer's human resources AI app, hence the name, and the game allows allows users to compete for spots on a leaderboard by answering questions based on 1 of 20 categories. I used Passport and JWT for authentication to keep everyone honest while allowing users to login with their Google or Facebook credentials. The front end is built with Next.js and deployed with Vercel. While the Node backend utilizes Sequelize ORM with a PostgreSQL database.
